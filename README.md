# Real Time Color Detection
This script allows the user to enter an image. The color presented 
in this image will be detected upon the ***Hue***, ***Saturation***, and
***Value*** selected by the user.

### Example
![example](result.gif)

### Running the script
```commandline
python detector.py
```

### Libraries used
- OpenCV
- Numpy
